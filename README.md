# chippy

CHIP-8 emulator implemented in Go.

### Installation

1. Clone the repository
2. Install project dependencies

```
go mod download
```

### How To Run

```
go run main.go -file=path/to/rom
```

**Note:** run the following to get information about the program flags

```
go run main.go -h
```
